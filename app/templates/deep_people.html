Deep Analysis — Above-Average Risk Groups

Discover which combinations of age group, road user type, and injury level occur MORE frequently than the statistical average — revealing hidden high-risk patterns in Victorian road incident data.

This analysis uses a nested SQL query. The inner query calculates the average incident count across all demographic groups. The outer query then returns only those groups whose count exceeds that average — surfacing statistically significant risk patterns.

What does this mean?

This nested analysis compares each demographic group's incident frequency to the average frequency across all groups. By focusing on the groups above the average, we highlight those most likely to represent a significant risk and deserve further investigation.

Methodology Note

The query uses a nested structure. First, it calculates the average number of incidents per demographic group. Then it selects only those groups exceeding that average, ranking the most significant risks.

SELECT AGE_GROUP, ROAD_USER_TYPE, INJ_LEVEL, COUNT(*) AS incident_count
FROM PERSON p
JOIN ACCIDENT a USING (ACCIDENT_NO)
GROUP BY AGE_GROUP, ROAD_USER_TYPE, INJ_LEVEL
HAVING COUNT(*) > (
  SELECT AVG(group_count)
  FROM (
    SELECT COUNT(*) AS group_count
    FROM PERSON p2
    JOIN ACCIDENT a2 USING (ACCIDENT_NO)
    GROUP BY AGE_GROUP, ROAD_USER_TYPE, INJ_LEVEL
  ) AS avg_table
)
ORDER BY incident_count DESC
LIMIT 20;
